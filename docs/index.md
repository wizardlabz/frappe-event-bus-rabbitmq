# Frappe Event Bus — RabbitMQ Provider Documentation

End-user documentation for the RabbitMQ provider. For an overview and installation, see the [README](../README.md).

> 🚧 Documentation is being written alongside the app. This index will grow as features land.

## Planned sections

- **Installation** — installing the provider on top of Frappe Event Bus core
- **Connection** — configuring a RabbitMQ connection (host, vhost, credentials, TLS)
- **Destination** — exchanges, routing keys, queues, durability, publisher confirms
- **Examples** — publishing ERPNext events to exchanges and queues
- **Troubleshooting** — common broker and delivery issues
